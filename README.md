**Instalacion de entregable de la prueba**

Para ejecutar el entregable se debe seguir los siguientes pasos:

1.Iniciar sesion en github desde visual y posteriormente tambien en tu navegador

2.Inicar un repositorio desde la terminal de una carpeta vacia en vs

3.Utilizar el comando git init para dar incio al repositorio 

4.Utiliza el comando "**git clone https://github.com/TITOS-DEV/PD-M2-HTMLYCSS.git**"

5.Posterior a esto abrir la carpeta creada llama **PD-M2-HTMLYCSS**

6.En esta se encuentra todo el codigo html y css y ademas los recursos como icones e imagnes utilizados para la realizacion de este

Si deseas ejecutar la pagina demaner rapida solo debes ingresar al siguiente link:

**LINK DE PAGINA DESPLEGADA:**  https://titos-dev.github.io/PD-M2-HTMLYCSS/
